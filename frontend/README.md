# JANSETU Frontend

JANSETU (जनसेतु) - A modern civic issue reporting platform that connects citizens with government authorities.

## Features

- 🌐 Bilingual support (Hindi/English)
- 📱 Responsive design for all devices
- 🎨 Modern UI with smooth animations
- 🗺️ Location-based issue reporting
- 👤 User dashboard for tracking reports
- 🔐 Secure authentication system

## Tech Stack

- React 19+ with Vite
- Tailwind CSS for styling
- Framer Motion for animations
- React Router for navigation
- Axios for API calls
- Leaflet for maps

## Getting Started

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```
